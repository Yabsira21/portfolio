Amazon

Live Site
The Amazon redesign project focuses on reimagining the e-commerce experience with a modern and user-friendly approach. The goal is to enhance navigation, simplify product discovery, and create a more engaging interface for users. Core features include an intuitive search and filtering system, dynamic product listings, and an optimized checkout process to streamline online shopping.
This redesign emphasizes accessibility and responsiveness, ensuring the platform is usable across various devices and screen sizes. By rethinking the visual hierarchy and incorporating clean, minimalistic design principles, the project aims to make online shopping faster, easier, and more enjoyable for all users.

Sheger Bingo

Live Site
The bingo game project is an interactive multiplayer experience designed for up to 100 players. Users can participate in a fast-paced, engaging bingo game where they compete for rewards. The game features a dynamic interface with real-time updates, allowing players to track their progress and interact with the game seamlessly.
The design focuses on providing an intuitive user experience, with easy-to-read bingo cards and smooth game mechanics. Built with a responsive layout, the game ensures players have an enjoyable experience across various devices. This project combines frontend development with essential backend features to deliver an exciting, competitive environment for all participants.
